
# Sippin'iT

One stop for all your coffee cravings



## Features (Only Screens)

- Home Page
- Product Listing Page
- Filters By
   * Sort
   * Price
   * Brand
   * Ratings
- Cart Management
- Wishlist Management
- Authentication:
     * Sign-up
     * Login
     * Logout
- Search
- Address Management
    * Address listing page
    * New Address form
- Single Product Page






## Tech Stack

* HTML
* CSS
